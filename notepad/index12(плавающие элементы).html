git init Чтобы превратить обычную директорию в Git репозиторий и начать версионировать файлы в ней, просто запустите git init. Впервые мы продемонстрировали эту команду в главе Создание Git-репозитория на примере создания нового репозитория для последующей работы с ним. Мы немного поговорили о смене названия ветки по умолчанию с "master" на что-нибудь другое в главе Удалённые ветки. Мы использовали эту команду для создания чистого репозитория для работы на стороне сервера в главе Размещение голого репозитория на сервере.
